#### `.lowercased()`

```swift
let name = "John Cena"
print(name.lowercased())
// "john cena"
```

#### `.uppercased()`

```swift
let loud = "hello".uppercased()
print(loud)
// "HELLO"
```

- Works on both `Character` types and `String` types
- Useful for comparing case-insensitive strings or formatting text
