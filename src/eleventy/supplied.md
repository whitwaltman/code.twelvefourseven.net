All templates can access:
- `pkg`
- `pagination`
- `collections`
- `page` (current page)
- `eleventy`

In the page variable, the following values are ones I tend to use the most:
- `url`: `/current/page/myFile/`,
- `fileSlug`: `myFile`,
- `filePathStem`: `/current/page/myFile`
- `rawInput`: `<!doctype html>...</html>`
