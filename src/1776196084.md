Can use [template literal types](https://www.typescriptlang.org/docs/handbook/2/mapped-types.html) to create new property names from existing ones. Remaps keys using the `as` clause.

```ts
type Getters<Type> = {
    [Property in keyof Type as `get${Capitalize<string & Property>}`]: () => Type[Property]
};

interface Person {
    name: string;
    age: number;
    location: string;
}

type LazyPerson = Getters<Person>;
```

This is the resulting type alias

```ts
type LazyPerson = {
    getName: () => string;
    getAge: () => number;
    getLocation: () => string;
}
```
