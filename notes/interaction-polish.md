- [ ] Keyboard support (tab order makes sense)
- [ ] Visible focus styles (not removed)
- [ ] Hit targets >= 44px when possible
- [ ] Works with trackpad + touch (pointer events)
- [ ] `prefers-reduced-motion` supported
- [ ] Empty/loading/error states exist
- [ ] No layout shift surprises
- [ ] Esc closes modals/panels (if relevant)
