1. Open file / create new file

```
C-x C-f
```

2. Save file

```
C-x C-f
```

3. Escape / interrupt current command

```
C-g
```

## Org mode

1. Open agenda

```
C-c a
```

2. Insert date

```
C-c .
```

3. Insert time and date

```
C-u C-c .
```
